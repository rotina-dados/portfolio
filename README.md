# Rotina — Dados & Processos

Landing page institucional da Rotina, consultoria especializada em dados e automação de processos.

## Stack

- HTML + CSS + JS vanilla — sem dependências nem build step
- Fontes: Space Grotesk e DM Mono via Google Fonts

## Desenvolvimento

Abra `portfolio.html` diretamente no navegador ou sirva com qualquer servidor estático:

```bash
npx serve .
# ou
python -m http.server 8080
```

## Estrutura

```
portfolio/
└── portfolio.html   # página única com CSS e JS embutidos
```
